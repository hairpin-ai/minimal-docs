*Migrate to CRA
==============

Migrate Vite.js instance to CRA (create-react-app)

* * *

**Step 1:**

*   Please do not **delete/edit** any files before running the command.
*   Used in the root directory.

Example:

    Minimal_Typescript
      ├─ vite-ts
      ├─ starter-vite-ts

  

    cd vite-ts
    # OR
    cd starter-vite-ts

**Step 2:**

In folder `vite-ts` / `starter-vite-ts`.

Run:

    npx vite-to-cra@latest
